# Setup

1. > Open **Command Prompt**

2. > Install **django** python-libray in your local machine by using following commands:

```shell
> pip install django
> pip install pillow
> pip install mysql
> pip install razorpay
```

3. > Run the code in command prompt using **python manage.py runserver**
